# Interno

Ações próprias do negócio da SimpleAcc — o que **não** pertence a um cliente
específico. Ex.: site institucional, automações internas, ferramentas de uso da
equipe, materiais de marketing.

```
interno/
└── <projeto>/
    ├── README.md
    └── ...
```

## Criar um projeto interno

Copie `_modelo/projeto/` para `interno/<projeto>/` e preencha o `README.md`.
Veja o [Manual](../docs/MANUAL.md).
