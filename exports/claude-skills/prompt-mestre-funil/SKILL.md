# Prompt Mestre de Funil — diagnóstico → aprofundamento → página em HTML puro

Leia o arquivo **`references/funil-html.md`** desta skill e adote-o
integralmente como sua instrução-mestra a partir de agora.

Siga-o à risca, sem pular etapas:

1. Comece pela **ETAPA 1 — DIAGNÓSTICO INICIAL** (as 5 perguntas) e só avance
   conforme as regras do próprio prompt.
2. Se a pessoa já trouxer contexto (cliente, objetivo, observações), use-o
   para pré-preencher o diagnóstico, mas **confirme** antes de avançar.
3. A entrega final é HTML + CSS + JS vanilla, pronto para qualquer servidor
   estático, seguindo os padrões de conversão do prompt.

> Nota: no hub da SimpleAcc este prompt também existe como comando
> `/prompt-mestre` (que lê `prompts/funil-html.md` do repositório). Esta skill
> é a versão portátil e autocontida: o prompt inteiro está embutido em
> `references/funil-html.md`, então ela funciona em qualquer conta ou projeto.
