# LP Alivance Club

Landing page do programa **Alivance Club** de Rafael Granella.

## Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS v4

## Como rodar

```bash
npm install
npm run dev
```

## Deploy

Hospedado na **Vercel**. Configurações:

- **Root Directory:** `clientes/rafael-granella/funis/lp-alivance`
- **Framework:** Vite
- **Domínio:** `lp.rafaelgranella.com.br`

Para conectar o domínio ao Vercel:
1. Adicionar `lp.rafaelgranella.com.br` no painel do projeto na Vercel
2. Copiar o valor CNAME fornecido pela Vercel
3. No cPanel da Turbo Cloud → Zone Editor → Adicionar CNAME `lp → <valor-vercel>`
