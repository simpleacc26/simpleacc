# Clientes

Um diretório por cliente. Dentro de cada cliente, um diretório por projeto/entrega.
O **Interno** fica no topo pelo prefixo `0-`.

```
clientes/
└── <cliente>/
    ├── README.md          ← quem é, contatos, links, projetos ativos
    └── <projeto>/         ← cada entrega (site, quiz, automação, app...)
```

## Criar um cliente novo

Copie `_modelo/cliente/` para `clientes/<cliente>/` e preencha o `README.md`.
Use minúsculas, sem acento e com hífen no nome da pasta. Veja o
[Manual](../docs/MANUAL.md).
