# Cliente: Sabrina Siqueira (Instituto Sabrina Siqueira)

Base de conhecimento do cliente. Tudo que a SimpleAcc faz para ele acumula aqui.

## Estrutura
- `contexto/`   — quem é, oferta, ICP, análise de mercado, linguagem.
- `estrategia/` — estratégias, planos, diagnósticos.
- `copy/`       — copy de páginas e anúncios.
- `roteiros/`   — roteiros de vídeo (datados).
- `funis/`      — funis, landing pages, quizzes. **Ver `funis/automacao-make.md`.**
- `aprendizados.md` — o que funciona/não funciona.

## Ao trabalhar para este cliente
1. Leia `contexto/` e `aprendizados.md` antes de criar.
2. Rode `/prompt-mestre` para gerar funil/página/anúncios.
3. Salve na pasta certa e registre o aprendizado.
4. Branch `cliente/sabrina-siqueira/<assunto>` + PR.

## Contexto rápido
- **Oferta principal:** dois funis de quiz da Dra. Sabrina Cantarino:
  - **Institucional / Autismo** — método para a clínica atender pacientes
    autistas e faturar com esse nicho ("Descubra quanto você poderia faturar").
  - **Implantes** — funil voltado ao público de implantodontia.
- **Estrutura dos funis:** Anúncio → Quiz → Relatório (na própria página) →
  Botão para WhatsApp. O relatório é gerado **na página**; o Make só cuida de
  CRM (GoHighLevel) + planilha (Google Sheets). Modelo espelhado da Carol e José.
