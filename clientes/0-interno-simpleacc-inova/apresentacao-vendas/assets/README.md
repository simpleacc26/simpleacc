# Imagens da apresentação

O deck usa **placeholders** (blocos marcados `[ ... ]`) onde entram imagens
reais. Coloque os arquivos aqui e troque o placeholder pelo `<img>` no slide
correspondente (ou me peça para fazer a troca).

## Dica de formato

- Fotos: JPG/WebP, lado maior ~1600px, otimizadas (< 300 KB cada).
- Prints de depoimento: PNG/JPG legíveis; manter proporção parecida facilita o grid.
- Nomes em minúsculas, sem acento, com hífen (ex.: `depoimento-eliana.png`).
