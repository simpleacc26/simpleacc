# Marca — Simple Acc

Identidade visual da Simple. Fonte de verdade do brandbook.

- `brandbook.html` — Brandbook v3.0 (abra no navegador). Cores, logo, tipografia,
  tom de voz, vocabulário e tokens.

## Tokens rápidos (cor)
- Navy meia-noite `#0A1626` (base) · Azul profundo `#122B4A` · Azul aço `#3E5C8A` (acento)
- Aço claro `#8197B6` (hairlines/legendas) · Branco gelo `#EEF3FA` · Branco `#FFFFFF`

> Regra: navy domina, branco abre respiro, azul aço só pontua. Nunca azul vívido,
> nunca dourado/guru, zero emojis. Serifa de alto contraste nos títulos da marca.
